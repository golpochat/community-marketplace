# Notification Management (Admin)

> **Screen:** `/admin/notifications` · `/super-admin/notifications` · **Permission:** `manage_notifications`

## Capabilities

- List/edit notification templates
- Send targeted notification to user
- Role-based broadcast
- Provider health status
- Delivery logs

## Broadcast workflow

1. **Notifications** → **Broadcast**
2. Select audience (role filter)
3. Choose template + variables
4. Confirm send — rate limits apply
